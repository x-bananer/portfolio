# Portfolio

Personal portfolio built with React and Vite.  
Showcases my projects, skills, and background.

## Stack
React   
Vite  

## Features
Sections: Skills, Experience, About Me, Projects, Certifications  
Project cards with links to live demos
Resume download and contact links
Responsive layout

## Local setup

```bash
yarn install
yarn dev
```

If you don’t have Yarn installed, you can use npm:

```bash
npm install
npm run dev
```
