# Portfolio

Frontend portfolio built with React, TypeScript, Vite, and SCSS Modules.

Designed as a fast, clean single-page product that presents real commercial experience through selected projects, technical skills, and production-focused UI.

[Live Demo](https://x-bananer.github.io/portfolio)

## Stack

React 19 • TypeScript • Vite • SCSS Modules • ESLint

## Includes

- responsive single-page layout
- smooth section navigation
- reusable UI components
- selected commercial projects
- experience, skills, and certifications
- GitHub Pages deployment

## Run locally

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
yarn preview
```

## Deploy

```bash
yarn deploy
```
